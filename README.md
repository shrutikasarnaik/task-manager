Task Manager



A simple Task Manager web application built using HTML, CSS, and JavaScript.



\## Features

\- Add new tasks

\- Mark tasks as completed

\- Delete tasks

\- Simple and clean UI



\## Technologies Used

\- HTML

\- CSS

\- JavaScript



\## How to Run

1\. Download or clone this repository

2\. Open index.html in your browser
